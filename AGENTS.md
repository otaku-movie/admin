## д»Јз Ғи§„иҢғ

- д»Јз ҒеҮЎжҳҜеҸҜд»ҘеӨҚз”Ёзҡ„е°ҪйҮҸиҝӣиЎҢеӨҚз”Ё
- **и·Ҝз”ұи·іиҪ¬зҰҒжӯўжӢјжҺҘе®Ңж•ҙи·Ҝеҫ„**пјҢеҝ…йЎ»з»ҹдёҖйҖҡиҝҮ `@/config/router` жҡҙйңІзҡ„ `processPath` еӨ„зҗҶпјҡ
  - `processPath(pathName)` з”ЁдәҺж—  query и·іиҪ¬пјҢдҫӢеҰӮ `router.push(processPath('agreementList'))`гҖӮ
  - `processPath({ name, query })` з”ЁдәҺеёҰ query и·іиҪ¬пјҢдҫӢеҰӮ `router.push(processPath({ name: 'agreementDetail', query: { id: row.id } }))`гҖӮ
  - `name` еҸ–еҗҺз«ҜиҸңеҚ•иҝ”еӣһзҡ„ `pathName`пјҢз”ұ `processPath` иҮӘеҠЁжӢјжҺҘ `/${lng}` еүҚзјҖдёҺжҹҘиҜўдёІпјҢ**йҒҝе…Қ**жүӢеҶҷ ``/${lng}/...`` жҲ– ``?id=${id}`` иҝҷз§ҚеҶҷжі•гҖӮ
  - ж–°еўһйЎөйқўж—¶пјҢйңҖеҗҢжӯҘеңЁиҸңеҚ•з®ЎзҗҶ / еҗҺз«ҜиҸңеҚ•ж•°жҚ®йҮҢзҷ»и®°еҜ№еә”зҡ„ `pathName`пјҢеҗҰеҲҷ `processPath` дјҡеҸ–дёҚеҲ° `path` иҖҢйҖҖеҢ–дёә `/${lng}undefined`гҖӮ
